# TalentR

> Smarter Hiring. Fairer Opportunities.
> Built for the Talentbank Hackathon 2026.

---

## What it is

TalentR is a Career OS prototype covering three modules:

| Module | Audience | Description |
|---|---|---|
| **Fair Pay Engine** | Candidates | Check your salary against real Malaysian market data |
| **Smart Talent Matching** | Employers | Match candidates by trajectory, not just CV history |
| **Live Internship Market** | Universities | Centralised internship marketplace matched by skills |

---

## Tech Stack

- Pure HTML, CSS, JavaScript — no frameworks, no dependencies
- Fully static — runs in any browser, deployable anywhere

## Running Locally

Just open `index.html` in a browser — no build step needed.

```bash
# Or use a simple local server
npx serve .
# then visit http://localhost:3000
```

## Deploying

### GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://<username>.github.io/<repo-name>`

### Netlify (drag & drop)
1. Go to [netlify.com](https://netlify.com)
2. Drag the entire `talentr/` folder onto the deploy area
3. Done — you get a live URL instantly
